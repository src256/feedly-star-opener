Feedly Star Opener
==============

Feedly Star Opener for Chrome and Firefox.


## Environment

- Chrome
- Firefox

## Install

- Chrome: Drop feedly-star-opener.user.js to the Extensions page.
- Firefox: Drop feedly-star-opener.user.js to the Greasemonkey dashboard.
